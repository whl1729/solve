### 电脑配置购买 checklist

- [x] 主板是否有四个内存槽（出于拓展性的考虑）
- [x] 主板是否支持内存的 DDR 类型（比如 DDR5）
- [x] 主板是否支持 WIFI 和 蓝牙
- [x] CPU 针脚必须和主板匹配，比如 LGA1700
  - 我购买 CPU 与 主板一体的
- [x] `电源功率 >= (CPU + 显卡 + 50w) * 1.5`
  - `1000w > (125w + 320w + 50w) * 1.5 = 750w`
- [x] 散热器是否配有硅脂
- [x] 散热器的针脚也要和主板及机箱匹配（支持 LGA1700）
- [x] 机箱是否满足散热器限高（机箱支持的最大高度要大于散热器高度 3mm 以上）
  - G500A SE 风冷限高为 190 mm、冷排为 `360 mm / 280mm / 240mm`
  - VK GL360 冷排尺寸为 `397 mm * 120 mm * 27mm`
- [x] 机箱是否满足显卡尺寸（注意显卡限长参数）
  - G500A SE 支持的显卡最大长度为 435 mm、最大高度为 180 mm
  - 索泰 4080 super 的长度为 309 mm、高度为 139 mm
- [x] 机箱是否满足主板规格（若主板是 E-ATX，需要了解机箱支持什么规格的E-ATX）
  - G500A SE 支持 E-ATX
- [x] 机箱是否具有风扇位（水冷规格）
  - G500A SE 支持 7 个 120mm 或 140mm 的风扇位
- [x] 机箱是否具有硬盘位以及槽位数量是否满足需求
  - G500A SE 支持 `SSD * 9 + HDD * 10`
- [x] 机箱是否满足电源规格（ATX、SFX 电源）
  - G500A 支持电源长度小于 195 mm
  - Vertext 海韵 GX1000 ATX 3.0，尺寸为 `160mm * 150mm * 86mm`
- [x] 如果要竖装冷排，要搞清楚自己显卡的长度和竖装冷排以及前面板风扇是否会起冲突
  - 不需要竖装冷排
- [ ] 显示器刷新率是否与显卡匹配

### 组装电脑遇到的问题

- 开机
  - 组装完成后，我误把重启键当做电源键，导致按下后毫无反应。
  - 组装完成后，出现两次电脑休眠后直接关机，查看 win 11 系统日志，找到 `0x9F` 的报错，暂时没找到原因。

- 显卡
  - 组装完成后，在任务管理器中没有显示 RTX 4080 Super 的信息，原因是没有安装驱动。
  - 英伟达官网驱动下载地址：`<https://www.nvidia.cn/geforce/drivers/>`，注意选择 40series，并且不要选择带 notebook 后缀的（那个是笔记本驱动）
  - 当连接上 wifi 后，电脑会自动打开 MSI Center 软件，并弹出一系列驱动安装请求（包括显卡驱动），我选择全部来安装了。
  - 为方便验证显卡是否为新卡，应该保留显卡包装盒（因为上面有产品序列号），出于维权考虑，最好录个开箱视频。

- wifi
  - 组装完成后，只能扫描出手机热点，无法扫描出包括客厅 wifi 在内的其他 wifi。原因是没有接 wifi 天线。
  - 除了接 wifi 天线之外，其他解决方案是：在房间里放置 wifi 中继器，或者使用有线网络。
  - 测网速对比（上传速率 / 下载速率，单位：Mbps）
    - wifi 中继：64 / 71, 64 / 70, 71 / 68，77 / 80，85 / 68, 74 / 86
    - 有线：101 / 103，71 / 65，71 / 74, 97 / 73, 80 / 69, 79 / 80

- 外放
  - 我购买的主板没有自带外放，需要自行购买。我尝试使用天猫精灵作为外放，但是经常卡顿，最终购买了漫步者 M30 有线/蓝牙 音响。

- 性能跑分
  - Cinebench
  - 3DMark
    - Stress Tests: Frame rate stability: 98.2%, Number of loops: 20
    - FPS: 87.6
  - Fulmark
  - HWINFO

### CPU

[Gyusang 的建议][2]：

- CPU 选型
  - 高端生产力兼顾游戏：13700KF/13700F/14700KF
  - 生产力旗舰性能选择：13900KF、7950X
  - 生产力没必要上X3D系列，14900K和13900KS性价比也不高，睿频略高，多核性能提升不大，13900K和7950X就足够了，再往上就线程撕裂者和至强。
  - 游戏旗舰性能选择：14900K/13900KS、7950X3D、7800X3D

- Intel CPU 后缀含义
  - 无标：没有任何后缀，具备核显不可超频，例如 Core i7 11700
  - K标：后缀为K结尾，具备核显并且可以超频，例如 Core i7 11700K
  - KF标：后缀为KF结尾，不具备核显并但可以超频，例如 Core i7 11700KF
  - F标：后缀为F结尾，不具备核显并也不可以超频，例如 Core i7 11700F

### GPU

[Gyusang 的建议][4]:

- 选A卡还是选N卡
  - 个人建议，游戏方面有光追需求选N，无光追需求看具体的性价比，
  - 生态方面由于N卡一直保持领先，布局比A卡更广，生产力N卡在部分领域还是有明显优势的，如果CUDA生态圈生产力的还是选N卡吧

- 针对游戏玩家的推荐
  - 1080P分辨率显卡推荐：RTX 4060/RX 7600及以下
  - 2K分辨率显卡推荐：RTX 4060/RX 7600及以上
  - 4K分辨率显卡推荐：RTX 4070/RX 7800XT及以上

- NVIDIA的Tensor核心专为深度学习加速而设计，对于AI和机器学习任务非常重要。

- N卡一线品牌：华硕、微星、七彩虹（准一线）

### 主板

[Gyusang 的建议][5]:

- 选择主板主要考虑以下几点：
  - 主板供电（供电能满足CPU需求就不会对电脑性能有影响）
  - 拓展性（多内存/显卡插槽、硬盘插槽、IO接口等）
  - WIFI/蓝牙功能（需要这两个功能主板需要选择带WIFI后缀的版本）

> 备注：购买内存时也要考虑是否跟主板兼容。比如主板支不支持高频内存条。

- Intel 芯片组
  - 高端芯片组Z系列：Z790
  - 中端芯片组B系列：B760
  - Intel芯片组中，H和B系列不支持CPU超频，只有Z系列支持CPU超频

- 主板品牌以御三家为首：华硕、微星、技嘉

- 主板规格
  - ATX (Advanced Technology Extended)：标准尺寸，305毫米 x 244毫米
  - MATX (Micro-ATX)：ATX 的缩小版，244毫米 x 244毫米
  - ITX (Mini-ITX)：三种规格中最小的主板，170毫米 x 170毫米

### 内存

[Gyusang 的建议][6]:

- 考虑是否需要[超频][7]
  - 买内存条时，首先要考虑自己要不要超频，如果不考虑超频，只需要考虑选择哪个品牌，购买多大的内存，多少频率的内存即可。

- DDR4 和 DDR5 怎么选
  - 由于频率高的原因 DDR5 生产力性能一直强于 DDR4
  - 预算充足就直接上 D5

- 内存容量
  - 选 32G
  - 购买时注意购买2根组成双通道，同等条件下双通道效能大于单通道

- 内存频率
  - 购买高频内存需要进bios开启xmp，不然D4默频2666MHz运行，D5默认4800MHz运行
  - 超频党是选内存颗粒，不超频就是直接选频率

- 内存超频主要看颗粒
  - 内存超频，主要是看内存的颗粒，颗粒决定了时序和频率，其实颗粒也决定内存的上限，所以内存品牌其实没颗粒重要。
  - “内存颗粒”是中国港台地区对内存芯片的一种称呼（仅对内存）
  - DDR5 颗粒天梯：海力士A-die ＞ 海力士M-die ＞ 三星颗粒 ＞ 英睿达颗粒

### 电源

[Gyusang 的建议][11]:

- 电源额定功率
  - 对于 显卡 RTX 4080 SUPER + CPU Intel i9，电源额定功率应该为 850 W

- 电源转换率
  - 一般选择80PLUS铜牌认证以上的电源就可以了

- 全模组、非模组
  - 全模组电源的线都是可以拆卸的，你可以用什么线插什么线，机箱内部会很整洁，也可以自己购买定制线，让你的电脑更拉风。
  - 非模组电源的线是不可拆卸的，在装机过程中，非模组电源的线有很多是多余的，走线很难，机箱会有点不整洁，如果有强迫症会很难受。

- 电源品牌选择
  - 首选：海韵、安钛克、振华、全汉、台达、EVGA、酷冷至尊、长城
  - 次选：航嘉、鑫谷、先马

### 散热风扇

[Gyusang 的建议][14]:

- 什么时候需要散热风扇
  - 机箱风扇不是必要配件，可装可不装
  - 机箱内部温度过高，需要机箱风扇辅助散热，这类需求一般是高端配置，对散热要求较高。

- 散热风扇选购指南
  - 想要性能强就选猫扇，想要性能和性价比兼顾就选利民，不在乎噪音就买10来块的风扇

### 键盘

[数码老罗的推荐][16]:

- 国产轴适用情况
  - 金粉轴：线性轴，直上直下，手感回弹有力，非常适合打字，一款吹爆的轴体，闭眼选
  - BOX白轴：段落轴，声音清脆，段落感强，比青轴好用，不易累
  - 静音月白轴：段落轴，一个比较独特的轴，段落强，声音小
  - G黄轴：线性轴，直上直下，手感有点重，声音小、闷，类比红轴，自带厂润，比红轴顺滑
  - G银轴：线性轴，常用于游戏，触发快，声音小，比极速银轴稳定
  - G白Pro轴（拾光白轴）：线性轴，类比金粉轴，性价比更高，触底声音比金粉小，总体比金粉稍微逊色一点，适合打字
  - 凯华极速银轴：线性轴，常用于游戏，触发快，声音小，轴心不稳

- 一直纠结的话，就选茶轴，金粉轴。
