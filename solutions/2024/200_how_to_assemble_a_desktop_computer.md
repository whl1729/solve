## 解答

### 台式电脑的组成部分

组装一台台式电脑，需要以下组件：

- CPU
- GPU
- 主板
- 内存
- 硬盘
- 显示器
- 电源
- 机箱
- 散热器
- 散热风扇
- 键盘

### CPU

[Gyusang 的建议][2]：

- CPU 选型
  - 高端生产力兼顾游戏：13700KF/13700F/14700KF
  - 生产力旗舰性能选择：13900KF、7950X
  - 生产力没必要上X3D系列，14900K和13900KS性价比也不高，睿频略高，多核性能提升不大，13900K和7950X就足够了，再往上就线程撕裂者和至强。
  - 游戏旗舰性能选择：14900K/13900KS、7950X3D、7800X3D

- Intel CPU 后缀含义
  - 无标：没有任何后缀，具备核显不可超频，例如 Core i7 11700
  - K标：后缀为K结尾，具备核显并且可以超频，例如 Core i7 11700K
  - KF标：后缀为KF结尾，不具备核显并但可以超频，例如 Core i7 11700KF
  - F标：后缀为F结尾，不具备核显并也不可以超频，例如 Core i7 11700F

### GPU

[Gyusang 的建议][4]:

- 选A卡还是选N卡
  - 个人建议，游戏方面有光追需求选N，无光追需求看具体的性价比，
  - 生态方面由于N卡一直保持领先，布局比A卡更广，生产力N卡在部分领域还是有明显优势的，如果CUDA生态圈生产力的还是选N卡吧

- 针对游戏玩家的推荐
  - 1080P分辨率显卡推荐：RTX 4060/RX 7600及以下
  - 2K分辨率显卡推荐：RTX 4060/RX 7600及以上
  - 4K分辨率显卡推荐：RTX 4070/RX 7800XT及以上

- NVIDIA的Tensor核心专为深度学习加速而设计，对于AI和机器学习任务非常重要。

- N卡一线品牌：华硕、微星、七彩虹（准一线）

### 主板

[Gyusang 的建议][5]:

- 选择主板主要考虑以下几点：
  - 主板供电（供电能满足CPU需求就不会对电脑性能有影响）
  - 拓展性（多内存/显卡插槽、硬盘插槽、IO接口等）
  - WIFI/蓝牙功能（需要这两个功能主板需要选择带WIFI后缀的版本）

> 备注：购买内存时也要考虑是否跟主板兼容。比如主板支不支持高频内存条。

- Intel 芯片组
  - 高端芯片组Z系列：Z790
  - 中端芯片组B系列：B760
  - Intel芯片组中，H和B系列不支持CPU超频，只有Z系列支持CPU超频

- 主板品牌以御三家为首：华硕、微星、技嘉

- 主板规格
  - ATX (Advanced Technology Extended)：标准尺寸，305毫米 x 244毫米
  - MATX (Micro-ATX)：ATX 的缩小版，244毫米 x 244毫米
  - ITX (Mini-ITX)：三种规格中最小的主板，170毫米 x 170毫米

### 内存

[Gyusang 的建议][6]:

- 考虑是否需要[超频][7]
  - 买内存条时，首先要考虑自己要不要超频，如果不考虑超频，只需要考虑选择哪个品牌，购买多大的内存，多少频率的内存即可。

- DDR4 和 DDR5 怎么选
  - 由于频率高的原因 DDR5 生产力性能一直强于 DDR4
  - 预算充足就直接上 D5

- 内存容量
  - 选 32G
  - 购买时注意购买2根组成双通道，同等条件下双通道效能大于单通道

- 内存频率
  - 购买高频内存需要进bios开启xmp，不然D4默频2666MHz运行，D5默认4800MHz运行
  - 超频党是选内存颗粒，不超频就是直接选频率

- 内存超频主要看颗粒
  - 内存超频，主要是看内存的颗粒，颗粒决定了时序和频率，其实颗粒也决定内存的上限，所以内存品牌其实没颗粒重要。
  - “内存颗粒”是中国港台地区对内存芯片的一种称呼（仅对内存）
  - DDR5 颗粒天梯：海力士A-die ＞ 海力士M-die ＞ 三星颗粒 ＞ 英睿达颗粒

### 电源

[Gyusang 的建议][11]:

- 电源额定功率
  - 对于 显卡 RTX 4080 SUPER + CPU Intel i9，电源额定功率应该为 850 W

- 电源转换率
  - 一般选择80PLUS铜牌认证以上的电源就可以了

- 全模组、非模组
  - 全模组电源的线都是可以拆卸的，你可以用什么线插什么线，机箱内部会很整洁，也可以自己购买定制线，让你的电脑更拉风。
  - 非模组电源的线是不可拆卸的，在装机过程中，非模组电源的线有很多是多余的，走线很难，机箱会有点不整洁，如果有强迫症会很难受。

- 电源品牌选择
  - 首选：海韵、安钛克、振华、全汉、台达、EVGA、酷冷至尊、长城
  - 次选：航嘉、鑫谷、先马

### 散热器

[Gyusang 的建议][13]:

- 怎么选择散热器
  - 散热器解热功耗 ＞ CPU功耗
  - 360水冷 > 顶级风冷 ≈ 240水冷
  - R9、i7K、i9K：360水冷

### 散热风扇

[Gyusang 的建议][14]:

- 什么时候需要散热风扇
  - 机箱风扇不是必要配件，可装可不装
  - 机箱内部温度过高，需要机箱风扇辅助散热，这类需求一般是高端配置，对散热要求较高。

- 散热风扇选购指南
  - 想要性能强就选猫扇，想要性能和性价比兼顾就选利民，不在乎噪音就买10来块的风扇

### 显示器

[Gyusang 的建议][15]:

- 面板
  - TN面板：色彩差、响应时间快、可视度差、对比度差、容易泛白，适合电竞FPS发烧友选择。
  - VA面板：色彩好，响应时间差，可视度一般、对比度好、有拖影现象，曲面屏大多都是VA面板
  - IPS面板：色彩好、响应速度一般、可视度广、对比度一般、普遍漏光，目前游戏、设计显示器的主流面板

### 键盘

[数码老罗的推荐][16]:

- 国产轴适用情况
  - 金粉轴：线性轴，直上直下，手感回弹有力，非常适合打字，一款吹爆的轴体，闭眼选
  - BOX白轴：段落轴，声音清脆，段落感强，比青轴好用，不易累
  - 静音月白轴：段落轴，一个比较独特的轴，段落强，声音小
  - G黄轴：线性轴，直上直下，手感有点重，声音小、闷，类比红轴，自带厂润，比红轴顺滑
  - G银轴：线性轴，常用于游戏，触发快，声音小，比极速银轴稳定
  - G白Pro轴（拾光白轴）：线性轴，类比金粉轴，性价比更高，触底声音比金粉小，总体比金粉稍微逊色一点，适合打字
  - 凯华极速银轴：线性轴，常用于游戏，触发快，声音小，轴心不稳

- 一直纠结的话，就选茶轴，金粉轴。
